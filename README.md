# 🧮 Calculadora Aritmética com React

![Calculadora](images/calculator.png)

Uma calculadora aritmética desenvolvida com **Vite + React +
Styled-Components**, capaz de realizar operações básicas:

-   ➕ Soma
-   ➖ Subtração
-   ✖️ Multiplicação
-   ➗ Divisão

O projeto possui **design moderno**, interface intuitiva e layout
totalmente **responsivo**, adaptando-se a diferentes tamanhos de tela.

<br>

## 🚀 Tecnologias Utilizadas

![Stack](https://skillicons.dev/icons?i=vite,react,js)

<br>

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como **desafio prático** do:

**Curso de Formação React Developer - Módulo I: Fundamentos do React**\
Plataforma: [DIO.me](https://web.dio.me/)

O objetivo foi aplicar conceitos fundamentais como:

-   Componentização
-   Estados com `useState`
-   Manipulação de eventos
-   Renderização condicional
-   Estilização com Styled-Components

<br>

## 🖥️ Funcionalidades

-   Inserção dinâmica de números
-   Atualização do display em tempo real
-   Tratamento de operações sequenciais
-   Limpeza do visor
-   Interface responsiva para mobile e desktop

<br>

## 📦 Instalação e Execução

Clone o repositório:

``` bash
git clone https://github.com/DevJoaoVitorB/react-calculator.git
```

Acesse a pasta do projeto:

``` bash
cd calculator
```

Instale as dependências:

``` bash
npm install
```

Execute o projeto:

``` bash
npm run dev
```
